## 素材库（8部经典）

### 古代中国（2部）

#### 1. 《世说新语》
| 项目 | 内容 |
|------|------|
| 作者 | 刘义庆（南朝宋） |
| 体裁 | 笔记小说 |
| 篇幅 | 1130则故事 |
| 主题 | 人物风度、性情才智、玄学清谈 |

**可用话题：**
- 「性情与风度」：荀巨伯远道探望病友、王子猷雪夜访戴
- 「真诚与矫饰」：王蓝田食鸡子、谢安寓教于乐
- 「坚守与变通」：嵇中散临刑东市
- 「雅量与胸襟」：谢安淝水之战胜后神色如常

**独特优势：** 每则故事仅数十字，文言简洁，适合引用开场

---

#### 2. 《太平广记》
| 项目 | 内容 |
|------|------|
| 作者 | 李昉等（宋） |
| 体裁 | 小说总集 |
| 篇幅 | 500卷，收录古籍700余种 |
| 主题 | 神仙鬼怪、人事幽明、奇闻异事 |

**可用话题：**
- 「因果与轮回」：唐太宗地府还魂、杜子春炼药
- 「情义与背叛」：柳毅传书、霍小玉传
- 「命运与抗争」：崔玄微护花、韦固遇月老
- 「善良与回报」：孙叔敖双头蛇、舍人替母还金

**独特优势：** 故事性强，易于改编成具体事例；兼具文学性与趣味性

---

### 近代中国（2部）

#### 3. 《缘缘堂随笔》
| 项目 | 内容 |
|------|------|
| 作者 | 丰子恺（1898-1975） |
| 体裁 | 散文随笔 |
| 代表作 | 《给我的孩子们》《梦痕》《送考》 |

**可用话题：**
- 「童真与成长」：怀念童年的纯粹，反思成人世界的复杂
- 「艺术与生活」：猫、燕子，白鹅——平凡事物的诗意
- 「爱与离别」：母亲的爱、与子女的温情
- 「初心与坚守」：在动荡中保持艺术家的本心

**独特优势：** 文字清新质朴，画面感强，易于仿写

---

#### 4. 《雅舍小品》
| 项目 | 内容 |
|------|------|
| 作者 | 梁实秋（1903-1987） |
| 体裁 | 散文随笔 |
| 代表作 | 《雅舍》《骂人的艺术》《时间即生命》 |

**可用话题：**
- 「生活美学」：在平凡中发现诗意
- 「批判与自省」：对世相百态的讽刺
- 「修养与境界」：一个人的修养体现在细节
- 「从容与焦虑」：在快节奏中保持内心平静

**独特优势：** 幽默睿智，既有文人雅趣，又有世俗洞察

### 现代中国（2部）

#### 5. 《我与地坛》
| 项目 | 内容 |
|------|------|
| 作者 | 史铁生（1951-2010） |
| 体裁 | 哲思散文 |
| 出版 | 2002年收入部编版语文教材 |

**可用话题：**
- 「生命与死亡」：地坛的启示，残疾与生命意义
- 「母爱与愧疚」：那个躲在角落里等我的母亲
- 「坚韧与挣扎」：在困境中寻找活下去的理由
- 「自然与心灵」：地坛不是废园，是心灵的庇护所

**独特优势：** 入选教材，情感深沉而不煽情，适合抒情议论

| 项目 | 内容 |
|------|------|
| 作者 | 王开岭（1969-） |
| 体裁 | 随笔杂文 |

## 使用方法

### 按话题查找（自动保存 HTML）

**格式：** `话题：xxx` 或直接说"查找 xxx 话题的素材"

**示例输入：**
```
话题：以退为进
```

**执行流程：**
1. 分析话题内涵
2. 从素材库中选择 7-8 个相关素材
3. 为每个素材提供：典故/故事、原文摘录、可写角度
4. 生成素材对比表
5. 提供开头/结尾用法
6. **自动保存为 HTML** 到 `D:/claude/作文材料/以退为进/以退为进.html`

**HTML 文件模板：**
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>作文素材 - {话题}</title>
    <style>/* 简洁的阅读样式 */</style>
</head>
<body>
    <header><h1>作文素材：{话题}</h1></header>
    <main>
        <!-- 话题解读 -->
        <!-- 7-8个素材详细解读 -->
        <!-- 素材对比表 -->
        <!-- 开头/结尾用法 -->
    </main>
</body>
</html>
```

## 引用技巧

- **开头引用法**：先讲故事/引句，再展开议论
- **结尾升华法**：结尾回扣素材，形成首尾呼应
- **对比论证法**：中外各一，中古各一，形成对比
